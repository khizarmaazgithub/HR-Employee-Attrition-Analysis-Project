🚀 HR Employee Attrition Analysis (Power BI)

📌 Project Overview

This project focuses on analyzing employee attrition data to identify key factors contributing to employee turnover. Using data analysis and visualization techniques, the project provides actionable insights to help organizations improve employee retention and workforce planning.

📃 Description

Employee attrition is a critical challenge for organizations as it impacts productivity, cost, and team stability. This project analyzes HR data to uncover patterns and trends behind employee turnover.

The analysis is performed using Excel, Python, and Power BI. An interactive dashboard is built to visualize important metrics such as attrition rate, employee demographics, job satisfaction, and salary distribution.

The dashboard enables HR teams to:

Identify high-risk employee groups
Understand key drivers of attrition
Make data-driven decisions to improve retention

🎯 Project Goals
Analyze employee attrition patterns
Identify key factors affecting turnover
Build an interactive Power BI dashboard
Provide actionable business insights

🧹 Data Cleaning & Preparation
Removed duplicate records
Handled missing values

Prepared data for dashboard visualization

📈 Dashboard Insights

Key insights derived from analysis:

Employees working overtime show higher attrition
Lower salary groups have higher turnover
Certain job roles have significantly higher attrition rates
Employees with low job satisfaction are more likely to leave
